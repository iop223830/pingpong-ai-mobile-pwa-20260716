# 乒乓智评 · 手机版（PWA）

本文件夹中的网页已封装为 PWA，可作为手机桌面应用安装，保留原型的配色、页面和交互。

## 发布前的文件

- `pingpong-ai-training-prototype.html`：应用主页面
- `manifest.webmanifest`、`sw.js`、`app-icon.svg`：安装图标与离线支持

请将以上三个文件与主页面保持在同一目录，部署到支持 HTTPS 的静态网页服务；不能只上传主页面。

## Android 安装

使用 Chrome 访问主页面网址，点浏览器菜单中的“安装应用”或“添加到主屏幕”。也可以点首页右上角的“安装 App”。

## iPhone 安装

使用 Safari 访问网址，点击“分享”按钮，选择“添加到主屏幕”。首次加载后，应用主页面会离线缓存。

## 说明

PWA 可像应用一样从手机桌面打开，但不需要应用商店上架。若需要 Android APK 或 iOS IPA，仍需分别进行原生打包、签名和发布。
